# Netflix-Content-Analysis
This project explores Netflix's content library using Tableau.
It analyzes:
- Growth of Netflix shows and movies added per year
- Most Common genres
- Top Countries contributing content

## Tools Used
- Tableau Public
- Excel (*for data cleaning*)

## Key Insights
- Netflix content grew rapidly between 2015 and 2019
- Dramas and Comedies are the top 2 genres that dominate the platform
- The United States and India are the countries which contribute the highest number of movies/shows

## About the Dataset
The dataset was gotten from Kaggle. It includes Netflix titles (i.e movies and shows) with fields such as genre, country, release year, and date added.
